# ポジショニング管理システム 概要

## 1. ソフトウェア名

ポジショニング管理システム

## 2. 概要

本ソフトウェアは、シフトマネージャーが日付・時間帯ごとにクルーを管理し、各クルーを適切なポジションへ配置するためのシステムである。

従来、紙や表計算ソフトなどで行っていたクルーの勤務時間管理やポジション配置を、画面上でまとめて操作できるようにすることで、配置作業の効率化を目的とする。

シフトマネージャーは、カレンダーから対象日を選択し、その日に勤務するクルーを登録・編集・削除する。その後、対象となる時間帯を選択し、その時間帯に出勤しているクルーをポジションへ配置する。作成したポジショニング情報は保存でき、必要に応じて前後の時間帯へ移動しながら配置を確認・編集できる。

## 3. 目的

本システムの目的は、クルーの勤務情報とポジション配置を一元的に管理し、シフトマネージャーの作業負担を軽減することである。

特に、以下の点を実現することを目的とする。

- 日付ごとにクルー情報を管理する
- クルーの出勤時間・退勤時間を登録する
- 時間帯ごとに出勤中のクルーを確認する
- クルーをポジションへ配置する
- 配置情報を保存し、後から確認・変更できるようにする

## 4. 対象ユーザー

本システムの主な対象ユーザーは、シフトやポジション配置を管理するシフトマネージャーである。

シフトマネージャーは、勤務するクルーの情報を登録し、時間帯ごとに必要なポジションへクルーを配置する役割を持つ。

## 5. 主な機能

### 5.1 日付選択機能

カレンダー画面から対象日を選択する機能である。選択された日付はシステム内に記録され、その日のクルー情報を管理する画面へ移動する。

### 5.2 クルー登録機能

選択した日付に勤務するクルーを登録する機能である。クルー名、出勤時間、退勤時間を入力し、登録することでクルー一覧に追加される。

### 5.3 クルー情報編集機能

登録済みのクルー情報を編集する機能である。クルー名や出勤時間、退勤時間を変更し、保存することでクルー情報を更新できる。

### 5.4 クルー情報削除機能

登録済みのクルー情報を削除する機能である。削除対象のクルーを選択し、確認後にクルー一覧から削除する。

### 5.5 時間帯選択機能

ポジション配置を行う対象の時間帯を選択する機能である。選択した時間帯に出勤しているクルーを抽出し、ポジショニング編集画面へ表示する。

### 5.6 ポジション配置機能

出勤中のクルーを各ポジションへ配置する機能である。クルーをドラッグ＆ドロップでポジション枠へ移動し、配置を画面上に反映する。

### 5.7 ポジション配置変更機能

すでに配置されているクルーを別のポジションへ移動する機能である。配置済みクルーを選択し、別ポジションへドラッグ＆ドロップすることで配置情報を更新する。

### 5.8 ポジショニング保存機能

作成したポジション配置情報を保存する機能である。保存ボタンを押すことで、現在の配置情報がシステムに保存される。

### 5.9 前後の時間帯への移動機能

ポジショニング編集画面から、前後の時間帯へ移動する機能である。時間帯を移動した際には、退勤したクルーを配置リストから除外し、新たに出勤するクルーを追加する。

## 6. 基本的な利用の流れ

本システムの基本的な利用の流れは以下の通りである。

1. シフトマネージャーがカレンダーから対象日を選択する
2. 選択した日付のクルー情報登録・編集画面を表示する
3. クルー名、出勤時間、退勤時間を登録する
4. 必要に応じてクルー情報を編集・削除する
5. ポジション配置を行う時間帯を選択する
6. 対象時間帯に出勤しているクルーを表示する
7. クルーを各ポジションへ配置する
8. 作成したポジショニング情報を保存する
9. 必要に応じて前後の時間帯へ移動し、配置を確認・変更する

## 7. 管理する主なデータ

本システムでは、主に以下のデータを扱う。

### Crew

クルー情報を表すデータである。

- クルーID
- クルー名
- 出勤時間
- 退勤時間
- 対象日

### DailySchedule

日付ごとのスケジュール情報を表すデータである。

- 日付
- 登録クルー情報

### TimeSlot

時間帯情報を表すデータである。

- 時間帯ID
- 開始時間
- 終了時間

### Position

ポジション情報を表すデータである。

- ポジションID
- ポジション名

### PositionAssignment

クルーの配置情報を表すデータである。

- 配置ID
- 対象日
- 時間帯ID
- クルーID
- ポジションID

## 8. 期待される効果

本システムを利用することで、以下の効果が期待できる。

- クルー情報とポジション配置を一元管理できる
- 時間帯ごとの出勤クルーを把握しやすくなる
- ドラッグ＆ドロップにより直感的に配置できる
- 配置情報の保存により、後から確認・修正しやすくなる
- 紙や表計算ソフトによる管理と比べて、作業ミスを減らしやすくなる

## 9. 今後の拡張案

今後の拡張案として、以下の機能が考えられる。

- 保存済みポジショニングの履歴表示
- CSV出力機能
- スマートフォンやタブレットでの操作対応

## 10. まとめ

ポジショニング管理システムは、日付ごとのクルー情報と時間帯ごとのポジション配置を管理するためのソフトウェアである。

シフトマネージャーは、クルーの登録・編集・削除を行い、対象時間帯に出勤しているクルーを各ポジションへ配置できる。これにより、ポジション作成作業の効率化と、配置情報の管理のしやすさを実現する。
